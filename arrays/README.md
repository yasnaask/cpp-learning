آرایه (array) یه جور جعبه‌ست که می‌تونی **چندتا مقدار از یه نوع** رو توش بذاری.  
مثلاً یه آرایه از عدد صحیح:

```cpp
int numbers[5] = {1, 2, 3, 4, 5};
