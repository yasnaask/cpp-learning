## 🤔 تابع چیه اصلاً؟

تابع یه تیکه کده که یه کار خاص انجام می‌ده.  
مثلاً تابعی بنویسیم که جمع دو عدد رو حساب کنه.  
بعد هر وقت جمع خواستیم، فقط همون تابع رو صدا بزنیم، بدون تکرار کد.

// tarif function
int sum(int a, int b) {
    return a + b;
}

// estefade az function

    cout << "Natije: " << result << endl;
